Kermiteの開発を行っています。

Kermiteとは?

自作キーボード用のソフトウェアです。GUIでの配列設定を前提としてファームウェアが設計されています。 専用のユーティリティソフトでキーマッピングを書き込んで使用します。

ファームウェア/ユーティリティソフトともに外部のソフトウェアになるべく依存しない作りにしています。 レイヤロジックやキーコードのマッピング処理なども、既存のものに頼らず独自に実装を行いました。

ファームウェアは自作キーボードでよく使われているProMicroに対応しています。また、最近発売されたProMicro RP2040への移行対応を行っています。

特徴

  • ユーティリティソフトでキーマッピングを変更できます。
  • 現在のレイヤに応じて表示が切り替わる、リアルタイムキーマッピング表示機能があります。
  • 背景を透過してデスクトップマスコットのように表示するウィジェット機能があります。
  • キーを追加したり、キーの位置を変更できるレイアウトエディタを内蔵しています。
  • GUI上での操作でMCUにファームウェアを書き込めます。
  • ファームウェアはC言語によるシンプルな実装で、カスタマイズが容易です。

動作環境

  • Windows 10
  • MacOS Catalina
